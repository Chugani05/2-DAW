## Instalación de FileZilla
Instalamos FileZilla mediante el comando:
```
apt install filezilla
```

La ejecutamos con el comando:
```
filezilla
```

Que nos abre la siguiente pestaña.
<div align=center>
    <img src="./imgs/img03.png" alt="3"/>
</div>
