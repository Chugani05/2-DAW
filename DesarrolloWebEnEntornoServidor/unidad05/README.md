## Proyecto
**Game Side** es una API desarrollada con Django que proporciona datos simulados para aplicaciones relacionadas con juegos. Permite acceder a información como títulos, géneros, plataformas y calificaciones, facilitando el desarrollo y prueba de funcionalidades en proyectos web. El código fuente está disponible en [Game Side](https://github.com/Chugani05-JesusLugo2002/GameSide.git).
